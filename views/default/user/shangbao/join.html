二、定义

1、商家保障服务：指您根据本协议约定的条款和条件及易赚其他公示规则的规定，通过易赚接手除"流量任务""收藏任务"以外的互动区中商保信誉任务（以下简称"商保任务"），应履行任务规定要求（留言提醒除外），在任务指定好评时限完成任务要求的来路进店方式，交易好评、动态评分、好评内容、指定收货地址、旺旺聊天 等任务内容；保障发布方在任务结束但未获得任务要求好评、接手方恶意维权、因接手人使用信用卡付款导致发布人被淘宝扣除手续费、任务交易导致发布方额外被扣除淘宝客佣金、因接手人问题交易遇到淘宝自查等情形下的权益。

2、保证金：指您根据本协议约定的条款和条件及易赚其他公示规则的规定缴存并冻结于您的易赚账户内相应存款，在您未履行商家保障服务承诺时用于对商保任务发布方进行赔付的资金。

3、先行赔付：指您与发布方通过易赚产生商保任务关系后，如因您未履行商家保障服务承诺而导致商保任务发布方权益或经济受损，且在发布方直接要求与您协商处理未果的情况下，易赚有权以易赚客服业务标准，根据相关证据材料和规则判定您是否应履行赔付义务。如是，则易赚有权自您的易赚账户直接扣除相应金额保证金款项赔付给发布方。

1、您同意遵守本协议、《易赚规章制度》、以及所有公示于易赚与"商家保障服务"相关的规则。

2、您拥有的权利和资格在易赚相关任务大厅接手商保任务。如因您的行为导致易赚遭受任何第三方提起的索赔、诉讼或行政责任，您应承担相应责任并使易赚免责。

3、您应在通过易赚接手商保任务过程中，应履行任务描述要求（留言提醒除外），在任务指定好评时限完成任务要求的来路进店方式，交易好评、动态评分、好评内容、指定收货地址、旺旺聊天 等任务内容；保障发布方在任务结束但未获得任务要求好评，接手方恶意维权、接手方使用淘宝客导致商家被扣佣金、接手人使用信用卡支付导致商家被扣手续费、或因接手人问题交易遇到淘宝自查等情形下的权益。

5、您同意当易赚受理发布方提出的商家保障服务维权和赔付申请时，您应积极配合，在易赚要求的时间期限内提供相关证据材料，以证明任务交易过程与结果不存在发布方提出的问题或符合双方的任务约定（任务留言要求除外），并保证所提交的证据材料真实有效。如易赚以易赚客服标准，对您和/或发布方提供的证据材料进行审核后，以其独立判断您未全面履行商家保障服务内容的，则易赚有权根据本协议及其它公示规则的规定，对因此遭受权益损失的发布方进行先行赔付。

四、保证金

您应根据以下条款之约定缴存，并不可撤销地授权易赚有权根据以下条款及易赚其他公示规则的规定冻结、使用、处置保证金。本条款之约定不适用于非易赚商保会员。

1、保证金的缴存

1.1 除平台发生过严重违规的用户外，您可在"会员中心"等相关页面自由选择主动缴纳或不缴纳商保保证金。

1.2 如您未根据上述1.1款之约定选择主动缴纳保证金，则出现以下任一情形时，您必须缴纳相应金额的保证金：

1.2.1 您违反《易赚规章制度》中的规定，包括但不限于因发生违规行为之"辱骂会员或客服"，"欺骗恐吓会员，恶意退款维权"，"接手任务曾发生淘宝客佣金"，"使用信用卡支付任务交易"，"旺旺恶意提及敏感词汇"，"中评，乱写评语，恶意评语，广告评语"任意一项，则易赚有权强制要求您三日内足额缴纳相应金额的保证金。否则易赚有权冻结账号，结算账户存款后终止您的一切服务；

1.3 尽管有上述约定，本协议有效期内，易赚有权根据您业务变化及实际赔付情况以书面方式通知您调整保证金金额。您应使保证金余额始终保持不低于本条款约定之金额，如发生赔付事件导致您需补缴保证金的，则易赚有权继续冻结您易赚帐户中的余额以补足保证金，如您的易赚账户余款不足，则您应在赔付事件发生之日起三日内补足。保证金不足期间，易赚有权暂停或终止向您提供全部或部分易赚服务。

1.4 不论前述何种情形，只要您事实上缴纳了保证金，保证金余额为足额时，您易赚账号的相关页面将出现"已交商户保证金"或类似文案之标识。

2、保证金的冻结

您不可撤销地授权易赚，将您根据本协议之约定缴存的保证金款项冻结于您的易赚账户中。

3、保证金的解冻

3.1 您需同时符合如下条件，方可申请解冻您根据本条1.1款之约定主动缴纳的保证金：

3.1.1 自您主动缴纳保证金之日起已满三个月；

3.1.2 距离最后一个接手的"商保任务"完成时间超过30天；

3.1.3 进行中已接手任务包括但不限于任务产生的纠纷均已处理完毕。

4.1 您同意，易赚有权在包括但不限于以下情况发生时，处置您的保证金：

4.1.1 如您在易赚接手商保任务、履行任务相关义务过程中，违反易赚任何规则或违反您对发布方的任务承诺，或被发布方申请索赔时，易赚有权根据其独立的判断，无须承担责任地，直接使用保证金对发布方进行先行赔付。

4.3 如您的保证金余额不足以赔付时，您应自行支付额外的赔付金额。在您保证金余额不足时，易赚没有使用自有资金向商保任务发布方或者任何第三方支付赔偿金、补偿金或其它任何款项的义务，您仍应对发布方或者其他权利人承担赔付、补偿义务。您明确同意如您的保证金余额不足以赔付，且您未自行向买家支付额外的赔付金额时，您不可撤销地委托易赚使用自有资金代您向买家进行赔付。不论是在本协议项下，易赚基于您的委托而代为赔付的合计总金额以400元人民币为上限。此等情形下，易赚有权向您追偿赔付金，包括但不限于采用以下两种方式：
